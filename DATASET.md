# Dataset Setup

This project requires two datasets.

---

## ASAP Dataset (Holistic Scoring)

Download from:
https://www.kaggle.com/competitions/asap-aes/data

Place file as:
data/training_set_rel3.tsv

---

## CELA Dataset (Parameter Scoring)
The CELA dataset is not distributed directly with this repository.
It is an academic dataset and must be obtained from its original research source or course distribution.
After obtaining the dataset, place the file as:

data/CELA.csv

---

Public mirrors of the dataset may exist online, but users are responsible for ensuring they follow the datasetï¿½s usage and licensing terms.
